HTML

  • The 'head' element contains information about the webpage.
  • The 'meta' elements tells the browser how to display the information on the webpage.
  • The 'body' element represents the visible content shown to the user.
  • class= and id= attributes identify elements and are used to selet them in CSS and JS.
  • 'ul' elements make unordered lists with 'li' (list) elements shown with bullet points
  • The 'link' element creates connection to the external file.
  • The 'rel' attribute specifies relationship between current and linked document resource.
  • The 'href' attribute specifies URL of external resource

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want aroudn the content inside an element.
  • display: block; - assigns image/element to start on a new line

Git

  • cd 'folder name': opens a folder
  • git status: checks what branch we are currently on
  • git checkout -b 'branch-name': creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m "{changes implemented}": assosiate a message with our commit
  • touch 'new file name': creates new folder within a directory
